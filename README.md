# Portfólio Marcelo Fuhr

Portfólio profissional desenvolvido para apresentar minha trajetória, projetos, currículo e formas de contato.

🔗 Site publicado: https://portfolio-marcelo-gamma.vercel.app

## Sobre o projeto

Este portfólio foi criado com foco em oportunidades profissionais nas áreas de tecnologia, suporte técnico, sistemas web, automações, processos digitais e soluções práticas para negócios.

A ideia do site é mostrar quem eu sou, o que estou construindo, minhas experiências e projetos desenvolvidos na prática.

## Páginas do portfólio

- Início
- Sobre
- Projetos
- Currículo
- Certificados
- Experiência
- Contato

## Projetos em destaque

### Trevo Barbearia

Sistema de gestão web desenvolvido para uma barbearia real, com foco em agendamento online, painel administrativo, serviços, produtos, relatórios e controle operacional.

> Projeto tecnicamente concluído e apresentado no portfólio com autorização. Não está em operação oficial.

### Sarmelo Delivery

Sistema de delivery inspirado em plataformas como iFood, com cardápio, carrinho, pedidos, autenticação, cupons e painel do restaurante.

Repositório do projeto: https://github.com/sarmelo01/Sistema-Delivery
